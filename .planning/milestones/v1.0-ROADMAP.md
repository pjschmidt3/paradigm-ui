**Status:** SHIPPED 2026-01-13

## Overview

Transform the existing component library into a distributable shadcn registry with free/premium tiers. Starting with registry infrastructure, then organizing components by tier, adding documentation, and finally building a marketing site to showcase the library.

## Milestone Summary

**Decimal Phases:**

**Key Decisions:**
- Freemium model with complexity-based tiers (basic free, composite paid)
- TailwindCSS v4 only — forward-looking, simpler maintenance
- Separate type files as registry lib items for proper installation
- shadcn-button naming to differentiate from premium button
- CodeBlock elevated from site component to premium registry component
