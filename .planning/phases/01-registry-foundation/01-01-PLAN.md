<research_summary>
shadcn registry schema (from Context7):

**registry.json structure:**
```json
{
  "$schema": "https://ui.shadcn.com/schema/registry.json",
  "name": "registry-name",
  "homepage": "https://example.com",
  "items": [...]
}
```

**Item properties:**
- `name`: unique identifier (kebab-case)
- `type`: "registry:ui" for UI components, "registry:component" for composite, "registry:hook" for hooks
- `title`: Human-readable name
- `description`: Component description
- `dependencies`: npm packages (e.g., ["motion", "lucide-react"])
- `registryDependencies`: other registry items this depends on (e.g., ["button"])
- `files`: array of file objects with `path` and `type`

**File types:**
- `registry:ui` - UI primitives
- `registry:component` - Composite components
- `registry:hook` - React hooks
- `registry:lib` - Utility libraries
</research_summary>
